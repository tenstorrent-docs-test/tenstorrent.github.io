ttnn.sub_bw

ttnn.sub_bw(input_tensor_a: ttnn.Tensor | ComplexTensor, input_tensor_b: ComplexTensor | ttnn.Tensor | int | float, *, are_required_outputs: List[bool] | None = [True, True], memory_config: ttnn.MemoryConfig | None = None, dtype: ttnn.DataType | None = None, activations: List[str] | None = None) → ttnn.Tensor or ComplexTensor

Performs backward operations for subtract of input_tensor_a and input_tensor_b or scalar with given grad_tensor. Supports broadcasting.

Args:

  • input_tensor_a (ComplexTensor or ttnn.Tensor)

  • input_tensor_b (ComplexTensor or ttnn.Tensor or Number): the tensor or number to add to input_tensor_a.

Keyword args:

  • are_required_outputs (Optional[bool]): required output gradients

  • memory_config (Optional[ttnn.MemoryConfig]): memory config for the output tensor

  • output_tensor (Optional[ttnn.Tensor]): preallocated output tensor

  • queue_id (Optional[uint8]): command queue id

Supported dtypes, layouts, and ranks:

Dtypes

Layouts

Ranks

BFLOAT16, BFLOAT8_B

ROW_MAJOR, TILE

2, 3, 4

Note : bfloat8_b/bfloat4_b is only supported on TILE_LAYOUT

Example:

>>> tensor1 = ttnn.to_device(ttnn.from_torch(torch.tensor((1, 2), dtype=torch.bfloat16)), device)
>>> tensor2 = ttnn.to_device(ttnn.from_torch(torch.tensor((0, 1), dtype=torch.bfloat16)), device)
>>> output = ttnn.sub_bw(tensor1, tensor2)
