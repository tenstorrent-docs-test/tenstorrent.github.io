ttnn.remainder

ttnn.remainder(input_a: ttnn.Tensor, input_b: ttnn.Tensor | float, *, memory_config: ttnn.MemoryConfig | None = None) → ttnn.Tensor

Perform an eltwise-modulus operation a - a.div(b, rounding_mode=floor) * b.”, “Support provided only for WH_B0.

Args:

  • input_a (ttnn.Tensor)

  • input_b (ttnn.Tensor or Number)

Keyword Args:

  • memory_config (Optional[ttnn.MemoryConfig]): Memory configuration for the operation.

Example::
>>> tensor = ttnn.from_torch(torch.tensor((1, 2), dtype=torch.bfloat16), device=device)
>>> output = ttnn.remainder(tensor1, tensor2)
